# Grade 7 Algebra Notes

Solving one-step equations: apply the inverse operation to both sides. Example: x + 3 = 10 → subtract 3 from both sides → x = 7.

Solving two-step equations: undo addition/subtraction first, then multiplication/division. Example: 2x + 3 = 11 → 2x = 8 → x = 4.

Substitution: replace the variable with its value. Example: if a = 3, then 4a + 2 = 14.
